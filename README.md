## Run Locally

Clone the project

```bash
  git clone https://github.com/CihatKOCAK/React-Portfolio-Website.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn start
```


## Features

- light/dark mode toggle (coming soon)

- responsive design
