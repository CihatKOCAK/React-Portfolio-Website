## Run Locally <a name="locally"></a>

Clone the project

```bash
  git clone https://github.com/CihatKOCAK/React-Portfolio-Website.git
```

Go to the project directory

```bash
  cd React-Portfolio-Website
```

Install dependencies

```bash
  yarn
  npm install
```

Start the server

```bash
  yarn start
```
